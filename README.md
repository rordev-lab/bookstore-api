* Ruby version : ruby-2.6.5 , Rails: 6

* Clone/download the application

   bundle install

* Database creation

  rake db:create

* Database initialization

  rake db:migrate

* Start the server

  rails server  

* How to run the test suite

  rails test

* List of api we have in the collection

  5. To search book
  6. To destroy/unpublish book
